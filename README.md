# vulkanese  
## It's Vulkan-Ease!  

This repository  
* imposes a hierarchical structure on Vulkan
* dramatically simplifies Vulkan usage
* is pure python
* runs SPIR-V compute shaders efficiently, across all modern GPUs
* makes compute shader debugging easy
